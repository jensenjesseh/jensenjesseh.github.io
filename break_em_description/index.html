Break 'em Game

This game was made over the course of two weeks. It was coded from scratch using JavaScript and HTML 5.

This project has the following characteristics:

  • Menu system
  • Variable paddle-ball reflection
  • Particle effects
  • High score system
  • Traditional breakout system mechanics

Note: The background artwork in this game is not original but from a creative commons source.
