Per-Fragment Diffuse and Specular Lighting and Reflection Mapping

This WebGL project has the following characteristics:

  • Renders a skybox
  • Reads and renders the Buddha Stanford ply file
  • Renders the model with per-fragment diffuse lighting
  • Renders the model with per-fragment diffuse lighting and per-fragment specular lighting
  • Renders the model with 100% per-fragment reflection mapping
  • Renders the model with 80% per-fragmnent diffuse and specular and 20% per-fragment reflection mapping
  • Uses perspective projection
